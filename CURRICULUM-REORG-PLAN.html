6 Risks & mitigations

  • Service-role secret broken (root of the enhance bug). Short-term: functions forward the admin JWT (done). Long-term: set the correct edge-function secret under the new key system — needed for any function with no user JWT (cron/webhooks).
  • Destructive Enhance overwrites edits + deletes the quiz. Mitigated this milestone (D6): Append/Merge is the default; Overwrite requires the confirm.
  • Production migration. Export first; ship a tested rollback.
  • Reorder concurrency / large lists. Batch updates; write only changed rows.
  • Course-player ordering regressions. Covered by the navigation Playwright test pre-deploy.
  • AI emits malformed interactive payloads. Server-side schema validation with reject/retry; Playwright asserts payload validity; legacy MCQ path is the fallback.

7 Rollback

  • DB: keep the pre-migration export; down drops domain_id + domains.
  • App: feature-flag the new Curriculum tab; old course-management UI stays until Phase 2 is signed off.
